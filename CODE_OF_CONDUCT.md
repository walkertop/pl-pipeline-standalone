# 贡献者行为准则

本项目采用 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 作为贡献者行为准则。

## 我们的承诺

为了营造开放和包容的环境，我们作为贡献者与维护者承诺：不论年龄、体型、可见或不可见的残疾、民族、性别特征、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人形象、种族、宗教、或性别取向，参与本项目及其社区都将是一次免于骚扰的体验。

## 我们的标准

有助于营造正面环境的行为示例包括：

- 对他人表现出善意与尊重
- 尊重不同的观点、立场与经历
- 给予和优雅地接受建设性反馈
- 对受我们行为影响的人承担责任，向他们致歉并从经验中学习
- 关注社区整体福祉，而不仅是个人利益

不可接受的行为示例包括：

- 使用性暗示语言或图像，以及任何形式的性骚扰
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公开他人的私人信息（物理或电子地址等）
- 在职业场合被合理视为不当的其它行为

## 执行责任

社区维护者负责澄清并执行可接受行为的标准，对任何他们认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正行动。

## 适用范围

本行为准则适用于项目的所有空间，也适用于个人在公开场合代表项目或其社区时。

## 执行

若遇到滥用、骚扰或其它不可接受的行为，请通过以下方式联系项目维护者报告：

- GitHub Issues (公开): https://github.com/walkertop/pl-pipeline/issues
- 私下邮件（敏感事件）: 请 @walkertop 于 Issue 中提出私聊需求

所有投诉都会得到及时和公正的审查和调查。
